Abstract

SSD-based approximate nearest neighbor search (ANNS) solutions alleviate the memory pressure by storing the large-scale index in SSDs, which facilitates billion-scale ANNS processing. However, these solutions fall short in the cost efficiency (i.e., QPS/$). To address this challenge, the authors propose TRIDENTANN, a co-design approach leveraging the heterogeneous hardware (SSD array, low-end GPU, and CPU), which achieves high-performance and cost-efficient ANNS. Specifically, TRIDENTANN searches the normal and noise clusters (i.e., sparse clusters isolated from dense clusters) separately via a hybrid cluster index, which enables efficient I/O utilization and parallel search operations. Additionally, TRIDENTANN customizes a user-space storage stack to fully utilize the high bandwidth between GPU and SSDs for efficient data transfer. Moreover, it orchestrates GPU-CPU-SSD task pipelines in parallel to maximize processing throughput. The evaluation results show that TRIDENTANN outperforms existing ANNS systems in terms of throughput, latency, and cost efficiency.
